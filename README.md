# horsebet

## Description
A Python Tkinter App which lets users put on bets on virtual horses with a virtual currency. These horses have odds and a random horse is selected each race

## Usage
-> User enters their age. They cannot enter if their age is under 18

-> User is prompted to login or register an account

-> User is directed to the home screen where they can view horses and their odds, logout and place bets

-> the logout button takes the user back to the login screen

-> Place Bet button prompts user to place their bet and choose a horse

-> their is a delay of a couple seconds then the winner is announced

-> if the player has won, their winnings will be shown in the messagebox

-> user is directed back to home page and results are saved to scorelog.txt
